# ECharts 在线演示平台

> 🎨 基于 Vue3 + ECharts 的交互式图表演示工具,支持实时编辑和预览

## ✨ 特性

- 🔄 **布局切换** - 支持上下布局和左右布局自由切换
- ↔️️ **拖动调整** - 鼠标拖动分割线,自由调整两个面板的比例
- 🎨 **渲染器切换** - 支持 Canvas 和 SVG 两种渲染方式实时切换
- ✏️ **实时编辑** - 直接在编辑器中修改 ECharts 配置
- ▶️ **一键渲染** - 点击按钮即可预览图表效果
- 📊 **示例加载** - 内置多种图表类型示例(折线图、柱状图、饼图等)
- 🎯 **错误提示** - 配置错误时显示详细的错误信息
- 📱 **响应式设计** - 完美适配桌面和移动设备

## 🚀 快速开始

```bash

```

## 📖 使用说明

### 1. 编辑配置

在左侧/上侧的配置编辑器中输入 ECharts 配置对象(JSON格式):

```json
{
  "title": {
    "text": "销售趋势",
    "left": "center"
  },
  "xAxis": {
    "type": "category",
    "data": ["周一", "周二", "周三"]
  },
  "yAxis": {
    "type": "value"
  },
  "series": [{
    "type": "line",
    "data": [150, 230, 224]
  }]
}
```

### 2. 渲染图表

点击 **"▶️ 渲染图表"** 按钮,右侧/下侧会立即显示图表效果。

### 3. 加载示例

从顶部工具栏的下拉菜单中选择预设示例(共18个):

**📊 基础图表**:
- 折线图 - 销售趋势分析
- 柱状图 - 季度销售对比
- 饼图 - 产品销售占比(环形)
- 散点图 - 身高体重关系
- 雷达图 - 产品能力对比

**🎯 高级图表**:
- K线图 - 股票行情展示
- 热力图 - 销售分布矩阵
- 仪表盘 - 任务完成率
- 树图 - 文件结构层级

**🔀 组合图表**:
- 柱线混合 - 双轴图表
- 堆叠柱状图 - 多维度分析

**📈 数据可视化**:
- 漏斗图 - 转化流程分析
- 桑基图 - 流量来源分析
- 平行坐标 - 多维数据分析

**🌟 特殊图表**:
- 关系图 - 知识图谱展示
- 箱线图 - 统计数据分布
- 主题河流 - 时间序列变化

### 3. 切换渲染器

在顶部工具栏选择渲染方式:
- **🎨 Canvas 渲染**: 适合大数据量,性能更好,默认选项
- **✨ SVG 渲染**: 矢量图形,放大不失真,适合打印

### 4. 拖动调整比例

将鼠标移动到两个面板之间的分割线上:
- **左右布局**: 鼠标变为 `↔️` 形状,水平拖动调整宽度比例
- **上下布局**: 鼠标变为 `↕️` 形状,垂直拖动调整高度比例

拖动范围: 20% - 80%,确保两个面板都有足够的显示空间。

### 5. 切换布局

点击 **"↕️ 上下布局"** 或 **"↔️ 左右布局"** 按钮切换界面布局。

### 6. 重置配置

点击 **"🔄 重置"** 按钮恢复到默认配置。

## 💡 使用技巧

### 1. 常用配置项

```json
{
  "title": { "text": "标题" },           // 标题
  "tooltip": { "trigger": "axis" },      // 提示框
  "legend": { "data": ["系列1"] },       // 图例
  "xAxis": { "type": "category" },       // X轴
  "yAxis": { "type": "value" },          // Y轴
  "series": [{                            // 数据系列
    "type": "bar",                        // 图表类型
    "data": [1, 2, 3]                     // 数据
  }]
}
```

### 2. 图表类型

- `line` - 折线图
- `bar` - 柱状图
- `pie` - 饼图
- `scatter` - 散点图
- `radar` - 雷达图
- `map` - 地图
- `candlestick` - K线图

### 3. 颜色配置

```json
{
  "color": ["#5470c6", "#91cc75", "#fac858"],
  "series": [{
    "itemStyle": {
      "color": "#5470c6"
    }
  }]
}
```
